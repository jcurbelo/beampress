Beampress Tutorial

Simple Usage

Main Container
Wrap all presentation content into a container e.g main-container. This element will contains all the frames and slide-items. 
                                <div class="main-container">
                                     <!-- The frames (presentation content) go here -->
                                </div>
Defining frames
Frames are defined using the frame class. The flow of the presentation slides will have the order in which the frame elements are defined. 
                                <div class="main-container">
                                     <section class="frame">First Frame</section>
                                     <section class="frame">Second Frame</section>
                                     <!-- ... -->
                                     <section class="frame">Nth Frame</section>
                                </div>

Using slides

Frame slides
The slides intervals of each 'slide-item' are specified with the data-onslide attr. A 'slide-item' can have one or more intervals (separated by comma). This syntax have the same Beamer's interval notation. Note that if an element doesn't contain this attr will always be visible. 
                                <section class="frame">
                                    <h1>I'll be always visible!!!</h1>
                                    <span data-onslide="1-">I'll be present from slide 1 to the end.</span>
                                    <span data-onslide="2,4">I'll be present on slide 2 and 4.</span>
                                    <span data-onslide="1,2-4,7-">I'll be present from slide 1 to 4 and then from slide 7 to the end.</span>
                                    <span data-onslide="-3,5-8">I'll be present until slide 3 and then from slide 5 to 8.</span>
                                </section>
Note:
Showing/Hiding 'slide-items' using this way, are made by transforming css opacity attr. No animation is used.

Changing style
'Slide-items' css properties can be updated on a specific slide using a 'data-style' attr. This 'data-style' recieves a 'property' : 'value' dictionary e.g data-style="'font-weight':'bold'", which are the rules to be updated on the 'slide-item'. Combining data-onslide and data-style attrs, different slide effects can be done. 
                                <section class="frame">
                                    <ul>    
                                        <li data-onslide="1-"> 
                                            <span data-onslide="2-" data-style="'font-style':'italic'">
                                                I want to 
                                                <span data-onslide="3-" data-style="'color': 'red'">
                                                    change on slide 2 & 3
                                                </span>
                                            </span>
                                        </li>
                                        <li data-onslide="2-">
                                            Hello 
                                            <span data-onslide="3-" data-style="'color': '#0b79e5', 'font-weight':'bold'">  
                                                bold 
                                            </span>
                                        </li>
                                    </ul>
                                </section>

Using custom frame 'data-onslide'
Custom frame intervals can be defined using 'data-onslide' in the 'frame' element; with this way only the 'slide-items' that are contained into frame intervals are shown. 
                                <section class="frame" data-onslide="2, 4-6">
                                    <h1>I'll be always visible despite frame intervals!!!</h1>
                                    <span data-onslide="1">I'm never going to be visible :-( </span>
                                    <span data-onslide="2">Cheers from and only slide 2 </span>
                                    <span data-onslide="3-">I'll be present from slide 4</span>
                                    <span data-onslide="4-">If you've followed this tutorial</span>
                                    <span data-onslide="5-">Then</span>
                                    <span data-onslide="6-">You know the rest</span>
                                </section>

Using slides

Again frame
The same frame but with differents slides can be shown using 'data-againframe'. This attr has the following syntax: 'data-againframe=[frame-id] intervals', where intervals have the same format as 'data-onslide' attr. For further information read Beamer Documentation 
                                <section id="f-1" class="frame" data-onslide="2, 4-6">
                                    <h1>I'll be always visible despite frame intervals!!!</h1>
                                    <span data-onslide="1">Now i'm going to exists with the first againframe call </span>
                                    <span data-onslide="2">Cheers from and only slide 2 </span>
                                    <span data-onslide="3-">I'll be present from slide 4 and in the second againframe call</span>
                                    <span data-onslide="4-">If you've followed this tutorial</span>
                                    <span data-onslide="5-">Then</span>
                                    <span data-onslide="6-">You know the rest</span>
                                    <span data-onslide="7-">See ya!!!</span>
                                </section>
                                <section class="frame" data-againframe="[f-1] 1"></section>
                                <section class="frame" data-againframe="[f-1] 2"></section>
                                <section class="frame" data-againframe="[f-1] 4-7"></section>
