Escalamientos métricos para MDS

Introducción

Transformaciones de las proximidades
  • Las proximidades son valores escalados por razón (ratio-scaled values).
  • En las ciencias sociales solo el rank-order de las proximidades es considerado.
  • Las disimilaridades `delta_(ij)` son entonces reemplazadas por las disparidades `hat(d)_(ij)` (d-hats) en la función de Stress.
  • Si sólo el rank-order de las proximidades es considerado informativo, entonces las disparidades deben tener el mismo rank-order que las proximidades.
  • En este caso estamos hablando de MDS ordinal o no métrico.
  • Si las disparidades están relacionadas con las proximidades mediante una función continua, entonces se considera MDS métrico.

MDS Métrico

Stress con d-hats
`sigma_r(hat(d),X) = sum_(i<j)(w_(ij)(d_(ij)(X) - hat(d)_(ij)) ^ 2)`
Notación
  • `hat(d)` denota el vector de `s xx 1` de disparidades con `s = ((n),(2))`
  • `X` denota la matriz de configuración

MDS Absoluto

Fórmula
`sigma_r(hat(d),X) = sum_(i<j)(w_(ij)(d_(ij)(X) - p_(ij)) ^ 2)`
  • En este caso las proximidades son exactamente las distancias entre los puntos `i` y `j`.
  • Desde el punto de vista de su aplicación es irrelevante.

MDS de Razón

Fórmula
`sigma_r(hat(d),X) = sum_(i<j)(w_(ij)(d_(ij)(X) - bp_(ij)) ^ 2)`
  • En este caso las proximidades deben ser disimilaridades.
  • Es fácil de optimizar ya que al derivar `eta^2(X) + b^2eta_p^2 - 2b rho (X)` con respecto a `b` obtenemos `b = (rho(X))/(eta_p^2)`
  • Esto nos da la actualización de `b` en cada iteración.

Jorge R. Curbelo Fernández, Eduardo Campver

MDS de Intervalo

Fórmula
`hat(d)_(ij) = a + bp_(ij)`

Este modelo es una generalización del MDS de razón y se conoce como MDS lineal ya que son transformaciones lineales de los `p_(ij)`

  • Si los `p_(ij)` son disimilaridades entonces `b > 0`
  • Si los `p_(ij)` son similaridades entonces `b < 0`
  • La razón de cualquier par de disparidades debe ser igual a la razón de sus proximidades correspondientes.

MDS Logarítmico

Por supuesto que la función no tiene que ser necesariamente lineal.

Fórmula
`hat(d)_(ij) = a + b log(p_(ij))`

En general se puede escoger cualquier función continua.

Pero algunas de ellas han probado ser bastante efectivas.

MDS Exponencial y Polinomial

Exponencial
`hat(d)_(ij) = a + b exp(p_(ij))`
Polinomial
`hat(d)_(ij) = a + b p_(ij) + c p_(ij)^2`

Conclusiones

Problemas

Disparidades negativas pueden resultar usando alguno de estos modelos.

  • Usando el MDS logarítmico cuando `p_(ij) < 1` ya que `log(x) < 0` para `0 < x < 1`
  • También puede ocurrir usando MDS de intervalo
Soluciones
  1. Restringir las disparidades a valores positivos
  2. Utilizar la modificación del algoritmo SMACOF para lidiar con disparidades negativas.
